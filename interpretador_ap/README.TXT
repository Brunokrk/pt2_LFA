Esta ferramenta foi desenvolvida inteiramente na linguagem Python, para a execução:

    -Instalar python no seu sistema operacional (De preferencia 3.8.2): 
        https://www.python.org/downloads/release/python-382/
        
        -Marcar opcção "Adicionar python ao PATH"
    
    -Após instalar python, extrair o ZIP baixado

    -No seu cmd, ou powershell, navegar até o repositório onde o trabalho foi extraido

    -após navegar até o repositório de salvação, executar o comando "python main.py"

    -Se tudo foi feito corretamente, o programa deve executar.


    OBS: Caso queira mudar o autômato, é necessário manipular o arquivo "entrada.json", conforme explicado no vídeo
